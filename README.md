# 🏢 IncomeView — Inteligência Financeira, Dados e Automação

A **IncomeView** é uma empresa especializada em **engenharia de dados financeiros, automação corporativa e inteligência analítica**. Atuamos na construção de pipelines, integrações, modelos e estruturas de governança que conectam **Finanças, Dados e Estratégia**, elevando a maturidade operacional e decisória das organizações.
Nosso propósito é transformar dados complexos em clareza, precisão e impacto real no negócio — sempre com ética, rigor técnico e visão de longo prazo.

---

## 🚀 O que fazemos
A IncomeView atua em quatro pilares centrais:

### 1. Automação Financeira e Operacional
- Pipelines inteligentes para FP&A, Controladoria, Tesouraria e Operações
- Integrações com ERPs, CRMs e sistemas financeiros
- Redução de esforço manual e aumento de confiabilidade

### 2. Arquitetura e Engenharia de Dados
- Modelagem relacional e governança
- Pipelines Bronze → Silver → Gold
- Integração entre ambientes locais (WSL/PostgreSQL) e nuvem (ADLS/Fabric)
- Estruturas escaláveis para dados financeiros e operacionais

### 3. Business Intelligence e Storytelling Executivo
- Dashboards para performance, liquidez, inadimplência, vendas e governança
- Modelagem tabular e análises avançadas
- Narrativas orientadas à tomada de decisão

### 4. Governança e Controles
- Rastreabilidade ponta a ponta
- Padronização de processos
- Integridade e consistência dos dados corporativos

## 📂 Projetos em Destaque

### 🔵 parquetData — Pipeline Moderno (WSL → ADLS → Fabric)
Pipeline completo para ingestão, transformação e publicação de dados em Parquet, PostgreSQL e Microsoft Fabric Lakehouse.
Inclui: Bronze Pipeline validado, integração ADLS (Files/), CLI, Docker e CI/CD.
➡️ Repositório: https://github.com/IncomeView/parquetData
### 🟢 siengeAPI — Automação e Integrações com o Sienge
Arquitetura modular para automação financeira e operacional via APIs do Sienge.
➡️ Repositório: https://github.com/IncomeView/siengeAPI

Modelos analíticos para FP&A, vendas, inadimplência, performance e governança.
### 🟡 Power Query — ETL e Padronização de Dados (em construção)
Pipelines de transformação, limpeza e integração de dados para ambientes corporativos.
